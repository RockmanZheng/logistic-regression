# data directory

wine - original data directory

x-y.data - data from wines of type x and y

train-x-y.data - train data set from wines of type x and y

test-x-y.data - test data set from wines of type x and y

train-set - train data set from 2/3 of wine.data

test-set - test data set from other 1/3 of wine.data
